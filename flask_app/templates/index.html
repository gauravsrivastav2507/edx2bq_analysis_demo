Welcome to edX data analysis

This website is based on the edX to BigQuery workflow. You can use this website to view a demo of sample edX data, or analyze your own edX data by signing on to your Google BigQuery account.


Demo of edx data analysis



Notes:
  • This code/website is not suitable for touchscreen devices, according to our tests.
  • This code is NOT production-grade, and not intended to be used directly with the BigQuery data to interpret student learning. We are working on the code. Email anindyar@mit.edu if you'd like to be notified about the "new and improved" versions.
  • The sample dataset shown here is representative but fake.
  • An earlier version of this demo (along with other useful jupyter notebooks) is available from Vita Lampietti's Github account.
  • A secondary goal of this website is to generate discussions about what learning-analytics quantities are easily available on the edX platform, and how instructors/researchers could use such quantities to improve/study student learning.
